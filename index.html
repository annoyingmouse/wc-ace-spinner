<wc-ace-spinner/>

Demo here: https://codepen.io/annoyingmouse/pen/eYQGNGO

A simple spinner component.

Installation

<script type="module" src="https://unpkg.com/@annoyingmouse/wc-ace-spinner/wc-ace-spinner.js"></script>

Alternatively:

<script type="module" src="https://cdn.skypack.dev/@annoyingmouse/wc-ace-spinner/wc-ace-spinner.js"></script>

You can use it minified by adding using this instead:

<script type="module" src="[WHATEVER CDN]/@annoyingmouse/wc-ace-spinner/dist/wc-ace-spinner.min.js"></script>

Please remember to change [WHATEVER CDN] to your favourite CDN

Configuration

  • Adding a width attribute allow you change the size of the element - it defaults to 100px by 100px, and valid number will change this.
  • Adding a inner-colour attribute with a HEX value will allow you to set the colour of the inner arc
  • Adding a middle-colour attribute with a HEX value will allow you to set the colour of the middle arc
  • Adding a outer-colour attribute with a HEX value will allow you to set the colour of the outer arc
